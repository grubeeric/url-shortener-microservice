URL Shortener MicroService

This microservice will take a URL as a paramter and return a shortened URL that will redirect to the original in JSON format.

Example usage: https://little-url.herokuapp.com/new/https://www.google.com

Returns: { "original_url":"http://www.google.com", "short_url":"https://little-url.herokuapp.com/8170" }

Then going to https://little-url.herokuapp.com/8170

Will redirect to: http://www.google.com
